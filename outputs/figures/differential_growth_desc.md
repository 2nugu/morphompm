# Figure: differential_growth

## Purpose
Demonstrate that incompatible (differential) morphoelastic growth in MLS-MPM
produces residual stress and macroscopic bending — the morphogenesis regime that
free isotropic swelling (which relaxes to zero stress) cannot exhibit.

## Data Source
- File: differential_growth.csv
- Generated by: test_growth.exe (test T3), rendered by scripts/plot_growth.py
- Date: 2026-06-28

## Axes
- X: x position (m) — strip long axis
- Y: y position (m) — bilayer thickness / deflection direction
- Color: elastic log-strain magnitude ||eps_e|| (proportional to residual stress)
- Slice: |z - z_center| < 0.03 m

## Key Observations
Numerical observations only.
- Bilayer strip: bottom layer Fg = 1.10·I, top layer Fg = 1.50·I.
- After relaxation: mean ||eps_e|| = 0.0997 (vs free-swell floor ~0.0031).
- Midline mean-y [L, C, R] = [0.6861, 0.7267, 0.6861]; bend = 0.0406 m.
- Symmetric (L == R): no spurious rigid rotation.

## Reproduction
cmake --build build --config Release
./build/Release/test_growth.exe
